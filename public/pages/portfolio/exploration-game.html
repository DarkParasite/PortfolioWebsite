Exploration Game

Blueprint Unreal Engine

October 2022 - November 2022

My first big order in Fiverr. The game mechanics are similar to the game Deep Rock Galactic, so the player can explore different maps, collect items and sell them to buy new equipment and repeat the loop again. The shooting mechanics implemented were pretty basic, but I implemented buy, sell, arsenal, and inventory systems, which was the main focus of the project. The client strictly wanted me to only use blueprints, so this project helped me get familiar with blueprints because I have always preferred C++ to blueprints and I still do.
