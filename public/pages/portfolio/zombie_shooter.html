Zombie Shooter Game

C++ Blueprint AI Unreal Engine

September 2023

A YouTuber placed an order on Fiverr to make a zombie shooter game for his YouTube channel. So I put together a zombie game in a few days with the help of my old shooter mechanics. In this game there are infinite number of waves where the difficulty of killing the zombies increases every wave. Higher the wave more zombies spawn and more aggressive they are.

Things Implemented

Shooting Mechanics - Rifle and Hand Gun

Zombie Spawn Logic

Zombie AI

Basic Score System

Project IGI inspired Health UI

Animation Blueprint for Player and AI
